# Full Stack Developer Internship Assessment

## 🔧 Technologies
- React.js
- Node.js + Express
- Axios

## 📂 Folder Structure
- `frontend/` – React App
- `backend/` – Node.js Server

## 🛠️ Setup Instructions

### 1. Backend
cd backend
npm install
node index.js

### 2. Frontend
cd frontend
npm install
npm run dev
